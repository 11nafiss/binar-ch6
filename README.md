# Challenge Chapter 6

### Built With

-   ExpressJS
-   Sequelize
-   PostgresJS
-   Swagger Open API

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/11nafiss/binar-ch6.git
    ```
2. In terminal go to ORM folder
   ```sh
    cd orm
    ```
3. Install packages
    ```sh
    yarn install
    ```
4. Next Step

    ```sh
    - setting file database.js
    - yarn db:create; yarn db:migrate; yarn db:seed
    ```

5. Run

    ```sh
    yarn develop
    ```
